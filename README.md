# Tiny ray-tracing engine

![Lighted Spheres](./samples/lighted-spheres.png)

![Many Spheres](./samples/many-spheres.png)

# How to run

```shell
cargo run --release SCENE
```

Available scenes:
  - `spheres`
  - `many-spheres`
  - `lighted-spheres`
